# SketchResume 产品需求文档（PRD V1）

## 一、产品概述

### 1. 产品名称
SketchResume（草图简历）

### 2. 产品定位
一款面向中国大学生的 AI 简历工具：

> 通过草图 / 参考图 / 文本描述，自动生成高质量、可编辑的简历。

核心价值：
- 降低排版门槛
- 提供非模板化的简历设计
- 用“意图表达”替代“工具操作”

---

## 二、目标用户

### 主用户
- 中国大学生（大三 / 大四）
- 校招求职人群
- 不会排版但有审美需求

### 次用户
- 转码/转行人群
- 留学生
- 对模板不满意的用户

---

## 三、核心使用场景

### 场景1：参考图生成
用户上传一张简历截图 → 自动生成类似布局的简历

### 场景2：草图生成
用户画草图 → AI生成结构化简历

### 场景3：旧简历优化
上传Word/PDF → 自动重排版 + 美化

---

## 四、核心功能模块

---

### 1. 多模态输入模块

#### 功能
支持以下输入：

- 图片上传
  - 手绘草图
  - 简历截图
  - 设计参考图
- 文本描述
  - 示例：
    - “双栏，左边信息，右边项目”
    - “互联网校招风，简洁一点”
- 原图修改
  - 可以直接在原图上修改，圈出需要修改的地方（bbox定位），输入修改需求
- 表单填写
  - 姓名
  - 教育背景
  - 实习经历
  - 项目经历
  - 技能

---

### 2. 布局解析模块（AI）

#### 功能
解析用户输入，生成结构化布局：

输出内容：
- 页面结构（单栏/双栏）
- 模块划分
- 模块顺序
- 视觉优先级

#### 示例（DSL）

```json
{
  "layout": "two_column",
  "left_width": 0.3,
  "sections": [
    "profile",
    "education",
    "internship",
    "projects"
  ]
}
```
